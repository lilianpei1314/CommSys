# CommSys

🔍 **一站式校园社团活动管理平台**  
专为高校设计的数字化社团管理系统，打通「学生-社团-校方」三方协作流程，实现从活动发布、报名审核到数据统计的全生命周期管理。

✨ **核心亮点**  
- ✅ **多角色协同**：学生便捷报名活动，社团高效管理成员，校方实时监控全局数据
- ✅ **全流程闭环**：支持活动发布→报名→审核→签到→统计的完整场景
- ✅ **智能看板**：可视化呈现社团活跃度、成员增长趋势等12项关键指标
- ✅ **即时通知**：微信/短信/站内信三端实时推送重要动态
- ✅ **权限隔离**：三级角色权限体系保障数据安全

🏗️ **功能架构**
| 前台功能                | 后台功能                |
|-------------------------|-------------------------|
| 🎯 社团分类检索         | 🔑 多层级权限管理       |
| 📅 活动日历视图         | 📊 数据可视化分析       |
| 📝 在线报名系统         | 📋 成员审核工作流       |
| 👤 个人参与档案         | 📨 批量通知中心         |
| 🔔 智能提醒服务         | 🗂️ 全站内容管理         |
