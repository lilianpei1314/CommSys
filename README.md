# CommSys
## 一.用户角色划分
### 1.普通学生:  
  <p>查看社团/活动、报名、查看个人参与记录</p>
### 2.社团管理员:
  <p>管理本社团信息、发布活动、审核成员</p>
### 3.系统管理员:
  <p>管理所有社团、用户权限、数据统计</p>
## 二.详细功能清单
### 1.社团展示:
  <p>分类检索、详情页（成员/活动/介绍）</p>
### 2.活动管理:
  <p>发布/编辑活动、报名审核、签到管理</p>
### 3.成员系统:
  <p>加入申请、角色权限、联系方式</p>
### 4.通知中心:
  <p>系统通知、活动提醒、审批结果推送</p>
### 5.数据统计:
  <p>成员增长趋势、活动参与热力图</p>
## 三.核心功能模块
### 1.前台功能
  <p>社团展示<br>活动报名<br>个人中心</p>
### 2.后台功能
  <p>成员管理<br>活动管理<br>数据统计</p>
