# CommSys
#一.用户角色划分
1.普通学生:  
  查看社团/活动、报名、查看个人参与记录
2.社团管理员:
  管理本社团信息、发布活动、审核成员
3.系统管理员:
  管理所有社团、用户权限、数据统计
#二.详细功能清单
1.社团展示:
  分类检索、详情页（成员/活动/介绍）
2.活动管理:
  发布/编辑活动、报名审核、签到管理
3.成员系统:
  加入申请、角色权限、联系方式
4.通知中心:
  系统通知、活动提醒、审批结果推送
5.数据统计:
  成员增长趋势、活动参与热力图
